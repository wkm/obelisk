Features

  • gocircuit worker discovery through zookeeper
  • counter and value metrics
  • worker inspection through stacktraces
  • simplistic in-memory timeseries store (will be replaced with Vena, see below)

Near Future

  • LevelDB-backed timestore based on GoCircuit's Vena
  • timeseries aggregation across multiple workers
  • remote log streaming (a'la Scribe)
